#Code from capstone project Handy Pantry

HandPantry_allfunctions -> A script containing most of the functions used to keep track of
items in the Handy Pantry and store them correcting. It includes many features like keeping
track of expiry dates and favorite items. These functions were integrated with the UI to
execute commands.

fooditem_class -> needed for script above. Written by another group member.

inputOutputDemo -> Run this script to demonstrate some of the features of the program. This
is a text-based adventure.
